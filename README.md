# Curso-Javascript-FullStack
Curso intensivo de 8 semanas para pasar de la lógica básica al desarrollo web profesional con JavaScript. Aprenderás sintaxis, programación funcional, datos asíncronos y Node.js. Con un enfoque práctico, usarás HTML, CSS y MySQL para crear "ZonaTecno", un e-commerce real, dominando todo el ciclo de vida del proyecto desde el frontend al backend.
