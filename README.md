# Curso Intensivo de Desarrollo Web: JavaScript & Node.js

Bienvenido al repositorio oficial del programa educativo de 8 semanas. Este curso está diseñado para guiarte desde los fundamentos de la lógica de programación hasta el desarrollo profesional de aplicaciones web full-stack.

## Descripción del Curso
Este programa avanza desde la sintaxis básica y la programación funcional moderna hasta el manejo avanzado de datos asíncronos y la arquitectura del lado del servidor.

## 💻 Proyecto Central: "ZonaTecno"
A lo largo del curso, aplicaremos los conocimientos teóricos en un proyecto: **ZonaTecno**, una plataforma de comercio electrónico funcional. 

### Tecnologías Utilizadas:
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** Node.js, Express
* **Base de Datos:** MySQL
